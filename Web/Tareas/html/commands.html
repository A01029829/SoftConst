Comandos de la Terminal

Página de inicio del blog

Descripción de los comandos:

Un comando de la terminal es una instrucción que puedes ejecutar en la línea de comandos de Git-Bash para interactuar con Git, administrar archivos o navegar por el sistema.

Lista de comandos:

-Comando- -Descripción-
pwd muestra la ruta actual
ls lista archivos y carpetas
cd cambia de directorio
mkdir crea una nueva carpeta
rm elimina un archivo
mv mueve o renombra un archivo
clear limpia la terminal
git init inicializa un repositorio Git
git clone clona un repositorio remoto
git status muestra el estado del repositorio
git add agrega cambios al staging
git commit -m "msj" guarda cambios con un mensaje

¡Listo! Estos son los comandos más importantes para la terminal, así como los más usados en Git. Dominar estos comandos es clave para trabajar de manera eficiente en los proyectos.
