¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje que se utiliza para definir el estilo y apariencia de una página web escrita en HTML. Con CSS se pueden cambiar colores, fuentes diseños y la posición de los elementos en la pantalla.

¿Cómo funciona CSS?

CSS funciona a través de reglas que estpan compuestas por:

  1. Selectores: que indican a qué elemento(s) se aplicará el estilo.
  2. Propiedades: que son características que se pueden modificar (color, tamaño, estilo, bordes).
  3. Valores: que especifican el ajuste de cada propiedad

Diferencia entre selectores por Tipo, Clase e Identificador

-Selector- -Símbolo- -Afecta a: - -Ejemplo-
Tipo (sin símbolo) Todos los elementos del mismo tipo p { color: blue; }
Clase (.) Varios elementos de la misma clase .Clase { color: green; }
Identificador (#) Un único elemento con ese ID #ID { text-align: center; }
